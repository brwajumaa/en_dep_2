Introduction to linguistics

Course Description

Course Code: LIT12305

/

Credit Hours: 3

/

Prerequisite Course: LIT11300

/

Co-requisite Course: LIT2300


An Introduction to Linguistics provides a brief introduction to the basic components of linguistic study such as phonetics, phonology, morphology, syntax, semantics, and pragmatics. The course also presents topics that lie in the outer layer of the linguistic circle as language actively contributes to a large number of human activities such as psycholinguistics, language variation, and language change. It has to be mentioned that all the topics covered are not language-specific. However, English will be given as an example, along with other languages.
